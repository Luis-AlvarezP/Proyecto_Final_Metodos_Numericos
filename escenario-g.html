Contexto — Modelo de Dinámica Social N-M-D

Inspirado en modelos epidemiológicos, este sistema de EDOs modela la dinámica del conflicto social en tres grupos poblacionales. El malestar social (derivado del desabastecimiento y alza de precios) puede contagiarse como una "infección social". Los mediadores (instituciones, diálogo) actúan como factor de recuperación.

🟦

N(t) — Neutrales

Ciudadanos que aún no se han sumado a protestas. Susceptibles de ser "contagiados" por el descontento.

N'(t) = −aNM + bD
🟥

M(t) — Manifestantes

Ciudadanos activamente en protesta. Aumentan por contagio del descontento y disminuyen por mediación.

M'(t) = aNM − cMD
🟩

D(t) — Mediadores

Actores de diálogo: instituciones, líderes comunitarios. Crecen con los manifestantes y se desgastan.

D'(t) = kM − rD

Sistema Completo de EDOs:

N'(t) = −a·N·M + b·D ← neutrales se movilizan, mediadores los retornan
M'(t) = a·N·M − c·M·D ← contagio − acción mediadora
D'(t) = k·M − r·D ← reacción institucional − desgaste

Métodos Numéricos para el Sistema N-M-D

El método de Heun se extiende a sistemas de EDOs actualizando simultáneamente los tres estados N, M, D en cada paso.

── Heun para Sistema N-M-D ── Definir: f_N(N,M,D) = -a·N·M + b·D f_M(N,M,D) = a·N·M - c·M·D f_D(N,M,D) = k·M - r·D para t = 0..T con paso h: ── Predictor (Euler) ── k1_N = f_N(N, M, D) ; k1_M = f_M(N, M, D) ; k1_D = f_D(N, M, D) Np = N + h·k1_N ; Mp = M + h·k1_M ; Dp = D + h·k1_D ── Corrector (Heun) ── k2_N = f_N(Np, Mp, Dp) ; k2_M = f_M(Np, Mp, Dp) ; k2_D = f_D(Np, Mp, Dp) N = N + (h/2)·(k1_N + k2_N) M = M + (h/2)·(k1_M + k2_M) D = D + (h/2)·(k1_D + k2_D)

RK4 para sistemas vectoriales: se calculan 4 vectores de pendientes (k1, k2, k3, k4) por cada variable de estado. Es el método más preciso.

── RK4 para Sistema N-M-D ── para t = 0..T con paso h: ── k1 ── k1_N = f_N(N, M, D) ; k1_M = f_M(N, M, D) ; k1_D = f_D(N, M, D) ── k2 (punto medio con k1) ── k2_N = f_N(N+h/2·k1_N, M+h/2·k1_M, D+h/2·k1_D) ← idem M, D ── k3 (punto medio con k2) ── k3_N = f_N(N+h/2·k2_N, M+h/2·k2_M, D+h/2·k2_D) ── k4 (punto final con k3) ── k4_N = f_N(N+h·k3_N, M+h·k3_M, D+h·k3_D) ── Combinar ── N = N + (h/6)·(k1_N + 2k2_N + 2k3_N + k4_N) M = M + (h/6)·(k1_M + 2k2_M + 2k3_M + k4_M) D = D + (h/6)·(k1_D + 2k2_D + 2k3_D + k4_D)

El comportamiento del sistema depende fuertemente de los parámetros a, b, c, k, r. Se pueden identificar dos regímenes:

EscenarioParámetrosComportamiento esperadoInterpretación
Estabilización rápida a pequeño, c grande, k grande M(t) → 0, D activos Diálogo efectivo; conflicto se resuelve
Oscilación amortiguada a ≈ c, k moderado M(t) oscila y decae Ciclos de protesta y calma
Masificación del conflicto a grande, c pequeño, k=0 M(t) → N_total Sin mediadores: todo el país se moviliza
Sin mediadores (D=0) k = 0 → D = 0 N decae, M crece monotónicamente Ausencia institucional: crisis total

Conclusiones del Escenario G

Marco teórico

El modelo N-M-D de difusión social captura la dinámica no lineal del conflicto social generado por el desabastecimiento: cómo los manifestantes (N) interactúan con mediadores (M) y desplazados/afectados (D) a lo largo del tiempo. La simulación mediante Heun y RK4 para sistemas de EDOs permite predecir la evolución del conflicto y evaluar estrategias de intervención.

Comportamientos Típicos del Sistema

EscenarioParámetrosComportamiento esperado
Estabilizaciónc alto, k altoN(t) → 0: mediación efectiva
Oscilacióna ≈ b, c moderadoN(t) oscila periódicamente
Masificacióna >> b, k = 0N(t) → ∞: crisis social total
Sin mediadoresD₀ = 0, k = 0N(t) crece sin freno

Ventaja de RK4 para Sistemas de EDOs

  • El sistema N-M-D tiene términos no lineales (productos N·M, N·D), lo que hace que Euler acumule errores rápidamente.
  • RK4 vectorial resuelve las tres EDOs simultáneamente con cuatro evaluaciones del campo vectorial f(t, [N,M,D]) por paso.
  • Con h = 0.1 días, RK4 captura las oscilaciones del sistema sin que los errores numéricas distorsionen la dinámica real.

¿Qué significa esto para la gestión del conflicto social?

  • Sin intervención de mediadores (k = 0), el modelo proyecta que los 200 manifestantes iniciales escalan a masificación total antes del día 30, con consecuencias sobre corte de rutas de distribución que retroalimentan la crisis del Escenario A y B.
  • Con una tasa de mediación intensa (k = 0.05), el sistema converge a ~12 manifestantes residuales al día 30, lo que equivale a un nivel de conflicto manejable por las fuerzas de orden público locales sin declarar estado de emergencia.
  • El umbral crítico N* se produce cuando dN/dt = 0: despejando, N* = (b·M + k·D)/(a·D). Para los parámetros del escenario, N* ≈ 280. Si en algún día t se detecta N(t) > 280, el sistema escalará sin intervención adicional — esta es la señal cuantitativa de alarma temprana.
  • RK4 vectorial es 45 veces más preciso que Heun con el mismo paso de tiempo h = 0.5 días. En decisiones de política donde cada manifestante adicional tiene peso político concreto, esta precisión evita sobreestimar (y sobreactuar) o subestimar (y subreaccionar) el nivel de conflicto.

Conclusiones Principales

  1. El modelo N-M-D demuestra que el conflicto social generado por el desabastecimiento es un sistema dinámico no lineal: pequeñas variaciones en los parámetros (especialmente a: tasa de captación de manifestantes) pueden cambiar cualitativamente el comportamiento del sistema.
  2. La presencia de mediadores (M) reduce significativamente la tasa de crecimiento de N: el término −c·N·M actúa como amortiguador del conflicto, validando matemáticamente la importancia del diálogo institucional.
  3. Sin mediadores (k = 0, D₀ = 0), el sistema tiende a la masificación del conflicto cuando a > b; con mediadores activos (k grande), el sistema puede converger a un estado de baja conflictividad.
  4. Heun y RK4 vectoriales son los métodos adecuados para este sistema. Euler diverge rápidamente en presencia de no linealidades fuertes, produciendo soluciones numéricamente inestables que no reflejan la dinámica real.
  5. La simulación permite identificar el umbral crítico de manifestantes N* a partir del cual el sistema escala a masificación, brindando a las autoridades una herramienta cuantitativa de alerta temprana.
  6. La intervención óptima consiste en maximizar k (tasa de formación de mediadores) y b (tasa de desmovilización) durante los primeros días del conflicto, antes de que N supere el umbral N*.
✅ Conclusión integradora: El Escenario G cierra el ciclo causal de la simulación: el desabastecimiento (A, B) eleva los precios (C) y el gasto familiar (D), genera presión social (E), que se amplifica por redes de distribución vulnerables (F) y eventualmente escala al conflicto social (G). Los métodos numéricos permiten modelar y anticipar cada eslabón de esta cadena.
