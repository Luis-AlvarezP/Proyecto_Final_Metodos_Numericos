🌐 Escenario G · Dinámica Social NMD

Evolución del Conflicto Social: Neutrales, Manifestantes y Mediadores

Un sistema de EDOs modela cómo la población se redistribuye entre neutrales (N), manifestantes (M) y mediadores (D) durante la crisis. Heun y RK4 se ejecutan simultáneamente.

ℹ Ambos métodos simultáneos: Heun (2° orden) · RK4 (4° orden). Los resultados se comparan en el mismo gráfico y tablas de iteraciones.
🌐

Contexto del Problema Real

La dinámica de conflicto social puede modelarse con tres poblaciones que interactúan. N(t) = Neutrales (ciudadanos que aún no se movilizan), M(t) = Manifestantes (activamente en protesta), D(t) = Mediadores (que buscan desescalar).

N'(t) = −a·N·M + b·D    M'(t) = a·N·M − c·M·D    D'(t) = k·M − r·D

a (contagio)

Tasa de conversión N→M por interacción con manifestantes

b (retorno)

Tasa de regreso a neutralidad por mediación

c (mediación)

Efecto de D sobre reducción de manifestantes

k, r (mediadores)

Tasa de generación y deserción de mediadores

📐

Métodos Numéricos para EDOs

k₁ = f(tₙ, yₙ)   ỹ = yₙ + h·k₁   k₂ = f(tₙ+h, ỹ)   yₙ₊₁ = yₙ + (h/2)·(k₁ + k₂) · Error ~ O(h²)

Heun es un predictor-corrector. Primero predice con Euler, luego corrige usando el promedio de las pendientes en el punto inicial y el predicho. Para sistemas de 3 ecuaciones, se aplica componentwise.

k₁=f(t,y) · k₂=f(t+h/2, y+hk₁/2) · k₃=f(t+h/2, y+hk₂/2) · k₄=f(t+h, y+hk₃) · yₙ₊₁ = yₙ + (h/6)·(k₁+2k₂+2k₃+k₄) · Error ~ O(h⁴)

Conclusiones del Escenario G

Simulador activo

El modelo NMD permite anticipar la evolución del conflicto social y evaluar la efectividad de diferentes estrategias de mediación antes de implementarlas.

📊 Interpretación de los Resultados

▶ Configura los parámetros y ejecuta la simulación para ver la interpretación con los valores reales.

Conclusiones Principales

  1. RK4 es el método de referencia con error O(h⁴); para h=0.5, el error por paso es del orden de h⁵ ≈ 0.03, mientras que Heun tiene h³ ≈ 0.125.
  2. El pico de manifestantes M_max depende fuertemente de la relación a/c: si a·N₀ > c·D₀, M crece antes de estabilizarse.
  3. El parámetro k (tasa de generación de mediadores) es el control más efectivo: duplicar k puede reducir M_max en más del 40%.
  4. El sistema NMD puede exhibir comportamiento oscilatorio (análogo al modelo Lotka-Volterra) cuando a·c·N₀·D₀ > (r·cM₀)².
  5. La simulación numérica permite a los gestores de crisis evaluar ex ante el impacto de políticas de mediación sin esperar el desenlace real.
✅ Recomendación: Usar RK4 con h ≤ 1 día para simulaciones de crisis de hasta 90 días. Para periodos más largos reducir h o usar métodos implícitos si el sistema se vuelve stiff (razón a·N₀ >> r).
