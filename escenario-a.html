Optimización de la Red de Abastecimiento y Transporte

Simula cómo distribuir carburantes y alimentos desde plantas de acopio hacia distintas zonas de la ciudad. Compara cinco métodos numéricos.

📋

Contexto del Problema Real

En situaciones de crisis, la distribución de productos esenciales (carburante, alimentos, medicamentos) debe optimizarse considerando la capacidad de cada planta, la demanda de cada zona y posibles bloqueos de rutas. El sistema de ecuaciones lineales modela exactamente estas restricciones de balance y flujo.

Explicación de los Algoritmos

El método de Jacobi calcula cada componente de la nueva iteración usando exclusivamente los valores de la iteración anterior. Requiere que la diagonal sea dominante para garantizar convergencia.

xᵢ⁽ᵏ⁺¹⁾ = (bᵢ − Σⱼ≠ᵢ aᵢⱼ · xⱼ⁽ᵏ⁾) / aᵢᵢ
para k = 0, 1, 2, ..., maxIter: para i = 0 ..n-1: suma = bᵢ para j = 0 ..n-1 (j≠i): suma -= aᵢⱼ · x_viejo[j] ← solo valores de iteración k x_nuevo[i] = suma / aᵢᵢ error = max|x_nuevo − x_viejo| si error < ε: terminar ✓ convergió
⚠ Jacobi puede ser más lento que Gauss-Seidel. Converge si la matriz es estrictamente diagonalmente dominante.

Similar a Jacobi, pero usa inmediatamente los valores actualizados de la misma iteración. Generalmente converge en la mitad de iteraciones.

xᵢ⁽ᵏ⁺¹⁾ = (bᵢ − Σⱼ<ᵢ aᵢⱼ · xⱼ⁽ᵏ⁺¹⁾ − Σⱼ>ᵢ aᵢⱼ · xⱼ⁽ᵏ⁾) / aᵢᵢ
para k = 0, 1, 2, ..., maxIter: x_viejo = copia(x) para i = 0 ..n-1: suma = bᵢ para j = 0 ..n-1 (j≠i): suma -= aᵢⱼ · x[j] ← usa el x actualizado inmediatamente x[i] = suma / aᵢᵢ error = max|x − x_viejo| si error < ε: terminar

SOR (Successive Over-Relaxation) acelera Gauss-Seidel mediante un factor de relajación ω. Con ω óptimo puede reducir el número de iteraciones significativamente.

xᵢ⁽ᵏ⁺¹⁾ = (1−ω)·xᵢ⁽ᵏ⁾ + ω·(bᵢ − Σⱼ≠ᵢ aᵢⱼ·xⱼ) / aᵢᵢ
para k = 0, 1, 2, ..., maxIter: x_viejo = copia(x) para i = 0 ..n-1: gs = (bᵢ − Σⱼ≠ᵢ aᵢⱼ·x[j]) / aᵢᵢ ← valor Gauss-Seidel x[i] = (1−ω)·x[i] + ω·gs ← corrección con ω error = max|x − x_viejo| si error < ε: terminar

La descomposición LU factoriza A = L·U donde L es triangular inferior y U superior. Luego resuelve el sistema en dos etapas de sustitución directa. Método directo: obtiene solución exacta (sin iteraciones).

A = L · U → L·y = b (sustitución ↓) → U·x = y (sustitución ↑)
── Fase 1: Factorización (Doolittle) ── para i = 0..n-1: U[i][k] = A[i][k] - Σⱼ<ᵢ L[i][j]·U[j][k] (k ≥ i) L[k][i] = (A[k][i] - Σⱼ<ᵢ L[k][j]·U[j][i]) / U[i][i] (k > i) ── Fase 2: Sustitución hacia adelante L·y = b ── para i = 0..n-1: y[i] = b[i] - Σⱼ<ᵢ L[i][j]·y[j] ── Fase 3: Sustitución hacia atrás U·x = y ── para i = n-1..0: x[i] = (y[i] - Σⱼ>ᵢ U[i][j]·x[j]) / U[i][i]

El Gradiente Conjugado es ideal para matrices simétricas y positivas definidas (SPD). Minimiza iterativamente el residuo. Teóricamente converge en exactamente n pasos.

αₖ = (rₖᵀrₖ) / (pₖᵀApₖ) · xₖ₊₁ = xₖ + αₖpₖ · βₖ = (rₖ₊₁ᵀrₖ₊₁) / (rₖᵀrₖ)
── Inicialización ── r₀ = b − A·x₀ ; p₀ = r₀ ; x₀ = [0,...,0] para k = 0, 1, ...: Ap = A · pₖ α = (rₖᵀ·rₖ) / (pₖᵀ·Ap) ← tamaño de paso xₖ₊₁ = xₖ + α·pₖ ← actualizar solución rₖ₊₁ = rₖ − α·Ap ← actualizar residuo si ‖rₖ₊₁‖ < ε: terminar ✓ β = (rₖ₊₁ᵀ·rₖ₊₁) / (rₖᵀ·rₖ) ← coeficiente de dirección pₖ₊₁ = rₖ₊₁ + β·pₖ ← nueva dirección conjugada
ℹ Para usar Gradiente Conjugado, la matriz A debe ser simétrica y positiva definida (SPD). Si la matriz no cumple esto, el resultado puede ser incorrecto.

Conclusiones del Escenario A

La simulación de la red de abastecimiento mediante sistemas de ecuaciones lineales permitió comparar cinco métodos numéricos aplicados a un problema real de distribución de carburantes y alimentos en contexto de crisis. A continuación se sintetizan los hallazgos principales.

Métodos Iterativos (Jacobi, Gauss-Seidel, SOR)

  • Gauss-Seidel converge en aproximadamente la mitad de iteraciones que Jacobi, porque incorpora los valores actualizados dentro de la misma iteración.
  • SOR con ω ∈ (1, 2) puede superar a Gauss-Seidel; el ω óptimo depende del radio espectral de la matriz.
  • Todos requieren que A sea diagonalmente dominante para garantizar convergencia; condición que modela correctamente plantas con alta capacidad propia.

Métodos Directos (LU, Gradiente Conjugado)

  • La descomposición LU (Doolittle) con pivoteo parcial produce la solución exacta en una sola pasada; sin riesgo de divergencia.
  • El Gradiente Conjugado converge en a lo sumo n pasos teóricos; ideal para matrices simétricas y positivas definidas (SPD).
  • Para redes de distribución pequeñas (n ≤ 50), LU es la opción más robusta y directamente recomendable.

Conclusiones Principales

  1. El sistema A·x = b modela correctamente las restricciones de balance en la red de distribución: cada fila representa una restricción de capacidad y cada columna una zona de demanda.
  2. Ante un escenario de bloqueo de ruta (coeficiente aij = 0), la solución cambia significativamente para las zonas afectadas, evidenciando la fragilidad de la red bajo perturbaciones externas.
  3. La descomposición LU resultó ser el método de referencia: solución exacta sin iteraciones y residuo ‖Ax − b‖ prácticamente nulo. Es el método recomendado para la toma de decisiones en crisis.
  4. Gauss-Seidel y SOR son adecuados cuando la matriz es demasiado grande para factorizar; su convergencia es garantizada si la red tiene alta demanda propia (diagonal dominante).
  5. Los cinco métodos convergen a la misma solución cuando la matriz es bien condicionada, lo que valida la consistencia del modelo matemático con el problema real.
  6. El modelo numérico permite simular distintos escenarios de crisis (bloqueos, incremento de demanda, rutas alternativas) sin costos operativos, apoyando la planificación de emergencia de las autoridades.
✅ Recomendación final: Para redes de hasta 4×4, usar Descomposición LU con pivoteo parcial. Para redes más grandes y matrices dispersas, Gauss-Seidel con ω ≈ 1.2–1.5 (SOR) ofrece el mejor equilibrio entre velocidad y precisión.
