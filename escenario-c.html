Desabastecimiento de Alimentos y Curva Continua de Precios

Reconstruye la curva de precios de productos básicos a partir de datos dispersos. Estima precios en días sin registro y analiza tendencias durante la crisis.

📋

Contexto del Problema Real

En mercados afectados por crisis de abastecimiento, los precios se registran de forma irregular: algunos días no hay datos, en otros el precio varía abruptamente. La interpolación permite reconstruir la curva continua de precios para analizar tendencias, detectar el momento de mayor inflación y estimar el precio en cualquier día del período.

Explicación de los Algoritmos de Interpolación

El polinomio de Lagrange interpola exactamente los n+1 puntos dados. Cada punto tiene su propio polinomio base Lᵢ(x) que vale 1 en xᵢ y 0 en todos los demás nodos.

P(x) = Σᵢ yᵢ · Lᵢ(x) donde Lᵢ(x) = Πⱼ≠ᵢ (x−xⱼ)/(xᵢ−xⱼ)
── Evaluación del Polinomio de Lagrange ── resultado = 0 para i = 0..n: Li = 1 ← base para el punto i para j = 0..n (j ≠ i): Li *= (x − xⱼ) / (xᵢ − xⱼ) resultado += yᵢ · Li retornar resultado ← = P(x*)
⚠ Con muchos puntos (n > 7), el polinomio de Lagrange puede producir oscilaciones numéricas entre nodos (Fenómeno de Runge). Splines cúbicos son más estables para datos dispersos.

La interpolación de Newton usa diferencias divididas para construir el polinomio de forma incremental. Agregar un nuevo punto solo requiere calcular una nueva diferencia.

P(x) = f[x₀] + f[x₀,x₁]·(x−x₀) + f[x₀,x₁,x₂]·(x−x₀)(x−x₁) + ...
── Tabla de Diferencias Divididas ── f[xᵢ] = yᵢ ← orden 0 f[xᵢ,xᵢ₊₁] = (f[xᵢ₊₁]−f[xᵢ]) / (xᵢ₊₁−xᵢ) ← orden 1 f[xᵢ,...,xᵢ₊ₖ] = (f[xᵢ₊₁,...,xᵢ₊ₖ] − f[xᵢ,...,xᵢ₊ₖ₋₁]) / (xᵢ₊ₖ−xᵢ) ── Evaluación en x* (método de Horner) ── P(x*) = coefs[n] para i = n-1..0: P(x*) = coefs[i] + P(x*)·(x*−xᵢ)

Los splines cúbicos naturales garantizan continuidad de la función y sus dos primeras derivadas en cada nodo. Producen curvas suaves sin oscilaciones y son ideales para datos de precios.

Sᵢ(x) = aᵢ + bᵢ(x−xᵢ) + cᵢ(x−xᵢ)² + dᵢ(x−xᵢ)³ para x ∈ [xᵢ, xᵢ₊₁]
── Spline Cúbico Natural (condición: M₀ = Mₙ = 0) ── 1. Calcular hᵢ = xᵢ₊₁ − xᵢ 2. Resolver sistema tridiagonal para Mᵢ (segundas derivadas): hᵢMᵢ + 2(hᵢ+hᵢ₊₁)Mᵢ₊₁ + hᵢ₊₁Mᵢ₊₂ = 6·[(yᵢ₊₂−yᵢ₊₁)/hᵢ₊₁ − (yᵢ₊₁−yᵢ)/hᵢ] → Algoritmo de Thomas (tridiagonal) 3. Calcular coeficientes de cada segmento: aᵢ = yᵢ bᵢ = (yᵢ₊₁−yᵢ)/hᵢ − hᵢ·(2Mᵢ+Mᵢ₊₁)/6 cᵢ = Mᵢ/2 dᵢ = (Mᵢ₊₁−Mᵢ)/(6hᵢ) 4. Evaluar en x*: encontrar intervalo [xᵢ,xᵢ₊₁] → Sᵢ(x*)
✅ Los splines cúbicos son la mejor opción para series de precios: evitan oscilaciones de Runge y producen curvas naturalmente suaves.

Conclusiones del Escenario C

La interpolación numérica permite reconstruir la curva continua de precios de productos básicos a partir de registros discontinuos, facilitando el análisis de la inflación durante períodos de desabastecimiento. Los tres métodos aplicados evidencian fortalezas y limitaciones distintas según el tipo de datos disponibles.

Lagrange vs. Newton (Diferencias Divididas)

  • Ambos construyen el mismo polinomio interpolante de grado n−1; la diferencia es solo algebraica.
  • Newton es computacionalmente más eficiente: agregar un nuevo punto de precio solo requiere calcular una nueva columna en la tabla de diferencias divididas.
  • Con más de 6 puntos ambos pueden presentar el fenómeno de Runge: oscilaciones espurias entre nodos que no corresponden a ningún precio real.

Splines Cúbicos Naturales

  • Garantizan continuidad C² en todos los nodos: la función, su derivada y su segunda derivada son continuas.
  • Producen curvas visualmente suaves que respetan mejor el comportamiento real de los precios.
  • El costo computacional es O(n) mediante el algoritmo de Thomas, significativamente más eficiente que los métodos globales O(n²).
  • Son la herramienta óptima para reconstruir series de precios con tendencia monótona.

📊 Interpretación de los Resultados

Conclusiones Principales

  1. Los tres métodos interpolaron correctamente los puntos de datos conocidos (error = 0 en los nodos), confirmando la solidez de los algoritmos implementados.
  2. Para pocos puntos (n ≤ 5), Lagrange y Newton ofrecen resultados equivalentes a los splines con menor esfuerzo de implementación; son útiles para estimaciones rápidas de campo.
  3. A medida que aumentan los puntos de datos registrados (n > 6), los splines cúbicos se vuelven claramente superiores: eliminan las oscilaciones de Runge y producen una curva de precios más realista y confiable.
  4. La diferencia numérica entre Lagrange y Newton es del orden de 10⁻¹⁰ a 10⁻¹² (errores de redondeo de punto flotante), confirmando que son algebraicamente idénticos.
  5. La extrapolación fuera del rango de datos es riesgosa con cualquier método: los precios fuera del período registrado pueden ser subvalorados o sobrestimados significativamente.
  6. Los momentos Mᵢ de los splines revelan las zonas de mayor tensión en la curva de precios: valores |Mᵢ| grandes indican cambios bruscos de precio, indicadores de posibles puntos de desabastecimiento agudo.
  7. La reconstrucción continua de la curva de precios permite calcular indicadores derivados: inflación diaria (primera derivada), aceleración del precio (segunda derivada) y gasto acumulado (integral, enlaza con el Escenario D).
✅ Recomendación final: Para monitoreo de precios durante crisis de abastecimiento, usar Splines Cúbicos Naturales con registros cada 2–5 días. Newton DD es preferible cuando los datos se registran de forma incremental (un producto nuevo por día) ya que no requiere recalcular toda la tabla.
