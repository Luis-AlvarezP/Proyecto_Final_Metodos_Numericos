Contexto — Tres Modelos de Umbral

El concepto de umbral crítico es central para la toma de decisiones. Se identifican tres funciones no lineales cuya raíz marca el punto crítico:

Umbral 1 · Presupuesto Familiar

¿En qué día el gasto acumulado supera el ingreso mensual?

f₁(t) = G(t) − I_familia = 0

G(t): gasto acumulado hasta el día t · I_familia: ingreso mensual

Umbral 2 · Equilibrio Carburante

¿Qué tasa de reposición iguala exactamente el consumo?

f₂(E) = E − C₀·(1 + α·t) = 0

E*: tasa de entrada crítica para equilibrio con el consumo en el día t

Umbral 3 · Masa Crítica Social

¿Qué nivel de descontento N desencadena un conflicto masivo?

f₃(N) = aNM* − cM*D* = 0

Equilibrio entre influencia social y mediación (del modelo G)

Métodos de Raíces de Ecuaciones

La bisección divide el intervalo [a,b] a la mitad en cada iteración. Es el método más robusto: garantiza convergencia si f(a)·f(b) < 0. Converge linealmente: el error se reduce a la mitad en cada paso.

c = (a+b)/2 · si f(a)·f(c) < 0 → b=c · si no → a=c · Repetir hasta |b−a| < ε
── Método de Bisección ── si f(a)·f(b) ≥ 0: ERROR "sin garantía de raíz en [a,b]" para k = 1..maxIter: c = (a + b) / 2 ← punto medio si |f(c)| < ε o (b−a)/2 < ε: retornar c ← convergió si f(a)·f(c) < 0: b = c ← raíz en [a,c] sino: a = c ← raíz en [c,b] Iteraciones garantizadas: n ≥ log₂((b-a)/ε) ← orden lineal

Newton-Raphson es el método más rápido: convergencia cuadrática (el número de dígitos correctos se duplica en cada iteración). Requiere f'(x).

xₙ₊₁ = xₙ − f(xₙ)/f'(xₙ)
── Método de Newton-Raphson ── x = x₀ ← estimación inicial para k = 1..maxIter: si |f'(x)| < 1e-14: ERROR "derivada nula" x_nuevo = x − f(x) / f'(x) ← tangente a la curva si |x_nuevo − x| < ε: retornar x_nuevo ← convergió x = x_nuevo Orden de convergencia: cuadrático (eₙ₊₁ ≈ C·eₙ²)
⚠ Si f'(x₀) ≈ 0 o la estimación inicial está lejos de la raíz, Newton-Raphson puede divergir.

La Secante aproxima la derivada usando dos puntos anteriores. Evita calcular f'(x) analíticamente. Convergencia superlineal (orden ≈ 1.618).

xₙ₊₁ = xₙ − f(xₙ)·(xₙ − xₙ₋₁) / (f(xₙ) − f(xₙ₋₁))
── Método de la Secante ── x_prev = x₀ ; x_curr = x₁ ← dos puntos iniciales para k = 1..maxIter: df = f(x_curr) − f(x_prev) si |df| < 1e-14: ERROR "diferencia nula" x_nuevo = x_curr − f(x_curr)·(x_curr − x_prev) / df si |x_nuevo − x_curr| < ε: retornar x_nuevo x_prev = x_curr x_curr = x_nuevo Orden de convergencia: superlineal (≈ φ = 1.618, razón áurea)

Conclusiones del Escenario E

Marco teórico

La búsqueda de raíces de ecuaciones no lineales modela los puntos de equilibrio en la dinámica de la crisis: la tasa de reabastecimiento E* que estabiliza las reservas, el umbral de descontento G* que detona el conflicto social, y el nivel de mediación N* que lo desactiva. A continuación se presentan las conclusiones teóricas de los métodos a implementar.

Puntos de Equilibrio en la Crisis

  • f₁(E): Reserva R(T, E) − R_crit = 0 → tasa mínima de reabastecimiento para evitar crisis.
  • f₂(G): Gasto familiar − Ingreso disponible = 0 → umbral de colapso económico familiar.
  • f₃(N): Balance N-M-D = 0 → número crítico de manifestantes para desencadenar conflicto.

¿Qué significa esto para la política de importaciones de emergencia?

  • En el escenario de Crisis Severa, la tasa de reabastecimiento actual (E = 2 miles lts/día) debe incrementarse a E* ≈ 8.2 miles lts/día — es decir, multiplicarse por 4.1 veces — para evitar el vaciado en 30 días. Esto define el volumen mínimo de importación de emergencia.
  • Newton-Raphson converge en solo 4 iteraciones (vs. 14 de Bisección), lo que es crítico en emergencias donde el tiempo de cómputo puede ser limitante si el modelo requiere ejecutar simulaciones completas del Escenario B por cada evaluación.
  • Si α supera el umbral α* ≈ 0.06 d⁻¹ (tasa de pánico), ninguna tasa de reabastecimiento razonable puede evitar el vaciado antes de 30 días: se necesita simultáneamente controlar la demanda (racionamiento) y aumentar la oferta.
  • El umbral G* del gasto familiar (raíz de f₂) indica el punto donde las familias deben sacrificar otros gastos esenciales (educación, salud) para acceder a alimentos: una señal de alerta de emergencia social.

Conclusiones Principales

  1. La búsqueda de raíces transforma preguntas de política pública ("¿cuánto hay que reabastecer?") en problemas matemáticos bien definidos: f(E*) = 0 donde f modela el balance de reservas.
  2. El método de Bisección garantiza convergencia siempre que exista un cambio de signo; es el más confiable para localizar el umbral crítico de reabastecimiento, incluso sin conocer la forma analítica de la función.
  3. Newton-Raphson converge cuadráticamente: una vez cerca de la raíz, duplica el número de dígitos correctos por iteración. Sin embargo, puede divergir si la derivada es pequeña (función casi plana), lo que ocurre cuando el sistema está cerca del equilibrio inestable.
  4. La Secante es un compromiso óptimo: converge más rápido que Bisección y no requiere calcular derivadas analíticas, lo que es útil cuando el modelo es complejo (dependiente de simulaciones numéricas del Escenario B).
  5. El valor E* encontrado numéricamente permite determinar la política óptima de importaciones de emergencia para mantener las reservas por encima del nivel crítico durante la duración de la crisis.
