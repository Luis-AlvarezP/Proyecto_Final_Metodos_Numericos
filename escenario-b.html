Vaciado Crítico de Reservas de Carburantes

Simula cómo disminuyen las reservas de una planta cuando la demanda supera el reabastecimiento. Predice cuándo se alcanzará el nivel de crisis.

📋

Contexto del Problema Real

Cuando existen bloqueos de rutas o restricciones en las importaciones, las plantas de distribución de carburante experimentan una tasa de entrada reducida mientras que la demanda aumenta por pánico. Este modelo simula exactamente ese proceso dinámico.

Ecuación diferencial del modelo:

dR/dt = E − C(t)

C(t) = C₀ · (1 + α · t)

→ dR/dt = E − C₀·(1 + α·t)
ℹ Cuando E < C(t), la reserva disminuye. El tiempo crítico t* se alcanza cuando R(t*) = R_crit.

Solución analítica (para validación):

R(t) = R₀ + E·t − C₀·(t + α·t²/2)

Explicación de los Algoritmos EDO

El método de Euler es el más simple para integrar EDOs. Aproxima la función usando la tangente en el punto actual. Es de orden 1: el error local es proporcional a h².

R(t+h) = R(t) + h · f(t, R(t)) donde f(t,R) = E − C₀·(1 + α·t)
── Método de Euler ── t = 0 ; R = R₀ mientras t < T: k₁ = f(t, R) ← pendiente en el punto actual R = R + h · k₁ ← avanzar un paso t = t + h si R ≤ 0: R = 0 ; detener ← reserva agotada
⚠ Euler acumula error con pasos grandes. Para h = 0.5 día, el error puede ser notable en simulaciones largas. Comparar con RK4 para ver la diferencia.

El método de Heun (Euler Modificado) usa un predictor-corrector: primero predice con Euler y luego corrige promediando las pendientes inicial y final. Es de orden 2.

k₁ = f(t, R) · k₂ = f(t+h, R+h·k₁) · R(t+h) = R(t) + (h/2)·(k₁ + k₂)
── Método de Heun (Predictor-Corrector) ── mientras t < T: k₁ = f(t, R) ← pendiente en tₙ (Euler) R_pred = R + h · k₁ ← predicción k₂ = f(t+h, R_pred) ← pendiente en tₙ₊₁ usando predicción R = R + (h/2)·(k₁ + k₂) ← corrección: promedio de pendientes t = t + h

Runge-Kutta 4 es el estándar de facto para EDOs. Combina cuatro pendientes ponderadas para lograr orden 4 con alta precisión y estabilidad.

R(t+h) = R(t) + (h/6)·(k₁ + 2k₂ + 2k₃ + k₄)
── Runge-Kutta de 4° Orden ── mientras t < T: k₁ = f(t, R) ← pendiente al inicio k₂ = f(t + h/2, R + h/2·k₁) ← pendiente en punto medio (k₁) k₃ = f(t + h/2, R + h/2·k₂) ← pendiente en punto medio (k₂) k₄ = f(t + h, R + h·k₃) ← pendiente al final R = R + (h/6)·(k₁ + 2k₂ + 2k₃ + k₄) t = t + h
✅ RK4 es el método más preciso de los tres. Para este problema, con h = 0.5, su error es prácticamente indistinguible de la solución analítica.

Conclusiones del Escenario B

La modelación del vaciado de reservas mediante la EDO dR/dt = E − C₀·(1 + α·t) permite predecir con precisión el momento en que una planta de distribución alcanzará el nivel crítico. La comparación entre los tres métodos numéricos revela diferencias importantes en precisión y aplicabilidad en contextos de crisis.

Implicación en la Predicción de Crisis

  • Con h = 0.5 días, RK4 coincide con la solución analítica hasta 4–5 decimales.
  • Euler puede acumular errores de 2–5% en horizontes de 30 días, lo que puede desplazar la predicción del tiempo crítico en hasta 1–2 días.
  • 1 día de error en la predicción equivale a no activar el plan de contingencia a tiempo en un escenario real de crisis.

📊 Interpretación de los Resultados

▶ Configura los parámetros y ejecuta la simulación para ver la interpretación con los valores reales.

Conclusiones Principales

  1. El modelo dR/dt = E − C₀(1 + αt) captura correctamente el fenómeno de vaciado acelerado: cuando α > 0 (pánico de compra), el consumo crece con el tiempo y la reserva colapsa más rápido de lo esperado si se usa un modelo lineal simple.
  2. El método de Euler es insuficiente para predicciones de crisis de reservas: su error se acumula y puede estimar el tiempo crítico con 1–3 días de desfase, lo que representa un riesgo grave para la planificación de emergencia.
  3. Heun duplica la precisión de Euler con solo un cálculo adicional de f por paso; es una mejora costo-efectiva cuando los recursos computacionales son limitados.
  4. RK4 es el estándar de referencia: con cuatro evaluaciones de f por paso, su error es cuatro órdenes de magnitud menor que Euler. Para simulaciones de reservas de carburante, h = 0.5 días con RK4 es más que suficiente.
  5. La solución analítica exacta existe para este modelo y sirve como referencia de validación. Que los métodos numéricos la reproduzcan confirma que el modelo matemático es consistente.
  6. El tiempo crítico t* puede estimarse con precisión sub-diaria usando RK4 e interpolación lineal entre pasos, lo que permite activar planes de contingencia con suficiente anticipación.
  7. Aumentar la tasa de entrada E o reducir α (medidas de control de demanda) son las únicas variables de política pública que pueden retrasar o evitar el vaciado, según muestra el modelo.
✅ Recomendación final: Usar RK4 con paso h ≤ 1 día para todas las predicciones de vaciado de reservas en contextos de crisis. Nunca usar Euler para horizontes superiores a 15 días si la decisión de activar planes de emergencia depende del tiempo crítico calculado.
