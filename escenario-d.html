💰 Escenario D · Integración Numérica

Costo Acumulado y Pérdida del Poder Adquisitivo Familiar

Calcula cuánto gasta una familia durante un mes cuando los precios suben progresivamente. Los tres métodos de integración se ejecutan simultáneamente.

ℹ Todos los métodos simultáneos: Trapecio · Simpson 1/3 · Simpson 3/8. La función C(t) = p(t)·q usa interpolación de Lagrange sobre los datos del producto.
📋

Contexto del Problema Real

El costo acumulado mensual es el área bajo la curva de gasto diario C(t) = p(t)·q, donde p(t) es el precio del producto en el día t y q la cantidad consumida. Compararlo con el ingreso familiar mide la pérdida del poder adquisitivo.

G_mes = ∫ₐᵇ p(t)·q dt ≈ método numérico  ·  PA = G_mes − p₀·q·(b−a)
ℹ La curva p(t) se obtiene mediante interpolación de Lagrange sobre los datos de precios registrados (Escenario C).
📐

Aplicación al Contexto de Crisis

  • G_crisis − G_normal estima la pérdida real de poder adquisitivo por producto.
  • Simpson 1/3 ofrece error < 0.1% para la curva de precios interpolada del Escenario C.
  • La suma de varios productos da el impacto total de la canasta básica.

Conclusiones Principales

  1. Los tres métodos de integración son aplicables a las curvas de precio continuas generadas por interpolación, donde C(t) = p(t)·q es suave y bien definida en todo el intervalo.
  2. La Regla del Trapecio es la más simple pero la menos precisa (O(h²)); sirve como línea de base de comparación.
  3. Simpson 1/3 reduce el error en cuatro órdenes de magnitud respecto al Trapecio con el mismo número de evaluaciones; es el método recomendado para esta aplicación.
  4. La integración numérica permite estimar el impacto monetario total de la crisis sobre una familia tipo, un indicador clave para políticas de subsidio y asistencia social de emergencia.
  5. La cadena analítica se completa: precios (C) → gasto acumulado (D) → umbral de colapso familiar (E).
✅ Recomendación: Usar Simpson 1/3 con n ≥ 30 para la curva de precios del Escenario C. El error de Simpson es < 0.01% mientras que el Trapecio puede llegar a 1–2% según la curvatura del precio.
