🔬 Escenario F · Número de Condición

Sensibilidad de las Redes de Distribución ante Perturbaciones

¿Cuánto error en el flujo esperado puede generar un pequeño error en la demanda medida? El número de condición κ(A) mide la amplificación del error en sistemas lineales.

ℹ Análisis de sensibilidad: Se calcula κ(A) = ‖A‖∞·‖A⁻¹‖∞ y se compara la cota teórica del error con el error real al perturbar b.
🔬

Contexto del Problema Real

El sistema Ax = b modela la distribución de recursos entre zonas durante la crisis. Si la demanda b tiene errores de medición δb, la solución x se perturba en δx. El número de condición κ(A) amplifica ese error: mayor κ(A) → mayor inestabilidad.

κ(A) = ‖A‖∞ · ‖A⁻¹‖∞  ·  Cota: ‖δx‖/‖x‖ ≤ κ(A) · ‖δb‖/‖b‖

κ ∈ [1, 30]

Sistema bien condicionado. Pequeños errores en b producen pequeños errores en x.

κ ∈ [100, 2000]

Sistema moderadamente mal condicionado. Los errores se amplifican notablemente.

κ > 5000

Sistema mal condicionado. La solución puede volverse completamente incorrecta con errores mínimos.

📐

Análisis del Número de Condición

‖A‖∞ = max_i Σ_j |aᵢⱼ|  ·  A⁻¹ por Gauss-Jordan  ·  κ(A) = ‖A‖∞ · ‖A⁻¹‖∞

La norma infinito de una matriz es la máxima suma de valores absolutos por fila. A⁻¹ se calcula mediante eliminación de Gauss-Jordan con la matriz identidad aumentada.

Ax=b → A(x+δx)=b+δb → Aδx=δb → δx=A⁻¹δb  ·  Error real: ‖δx‖∞/‖x‖∞

Se calcula la solución exacta x, luego se perturba b en ±δ% y se resuelve el sistema perturbado. La cota teórica es κ·(‖δb‖/‖b‖) y el error real es ‖x_pert − x‖/‖x‖.

📊 Interpretación de los Resultados

▶ Ingresa la matriz, el vector b y ejecuta el cálculo para ver la interpretación con los valores reales.

Conclusiones Principales

  1. El número de condición κ(A) es el principal indicador de fiabilidad de un sistema lineal; κ cercano a 1 es ideal, κ > 1000 indica problemas graves.
  2. Matrices diagonalmente dominantes (red balanceada) producen κ < 30; bloqueos parciales y redistribuciones forzadas pueden elevar κ a miles.
  3. Un κ(A) = 500 con un error de medición del 5% en b puede generar hasta un 2500% de error teórico en la solución x — inaceptable para políticas de distribución.
  4. El análisis de perturbación distingue entre la cota teórica (peor caso) y el error real (caso específico), que suele ser mucho menor.
  5. Durante la crisis, las redes de distribución con κ > 200 deben usar mediciones redundantes y técnicas de regularización para resultados confiables.
✅ Recomendación: Antes de confiar en la solución de un sistema Ax=b para distribuir recursos en crisis, calcular κ(A). Si κ > 100, revisar la redundancia de sensores de demanda y la representatividad del modelo de red.
